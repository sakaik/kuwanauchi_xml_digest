## このフォルダは
202504公開データに関する集計情報などを保管・公開するためのものです。


## 公開ファイル
### 地区別202404 to 202504 変更点
- Diff2024to2025.pdf: 地区別の対照表
  - 各データのデリミタは "/" です
- Diff2024to2025.zip: 上記PDFファイルの元ネタとなるテキストファイル（何かの処理に使いたい方はこちらが便利です）
  - 先頭が "-" のものが202404データのみに存在するもの
  - 先頭が "+" のものが202505データのみに存在するもの
